План разбивки задачи на подзадачи (C++ проект: поиск и замена в JSON файлах)

------------------------------------------------------------
1. Генерация JSON-файлов (модуль generator)
------------------------------------------------------------
1.1. Создать структуру FileObject { name, content }  
1.2. Реализовать функцию генерации случайного текста  
1.3. Реализовать добавление ключевых слов для поиска  
1.4. Реализовать сохранение массива объектов в JSON-файл  
1.5. Вывод диагностической информации (сколько файлов создано)

------------------------------------------------------------
2. Загрузка JSON (модуль json_loader)
------------------------------------------------------------
2.1. Загрузить JSON в память  
2.2. Преобразовать JSON в vector<FileObject>  
2.3. Обработка ошибок: нет файла, пустой файл, некорректный формат

------------------------------------------------------------
3. Поиск подстроки (модуль finder)
------------------------------------------------------------
3.1. Реализовать подсчёт вхождений строки в строку  
3.2. Режим с учётом регистра  
3.3. Режим без учёта регистра  
3.4. Тестирование на длинных строках

------------------------------------------------------------
4. Замена подстроки (модуль replacer)
------------------------------------------------------------
4.1. Функция replace_all  
4.2. Подсчёт количества замен  
4.3. Проверка на отсутствие совпадений  
4.4. Поддержка режима dry-run

------------------------------------------------------------
5. Формирование отчёта (модуль reporter)
------------------------------------------------------------
5.1. Структура отчёта ReportItem  
5.2. Формирование таблицы (CSV или консольный вывод)  
5.3. Сохранение в report.csv

------------------------------------------------------------
6. Запись результата в новый JSON (writer)
------------------------------------------------------------
6.1. Запись обновлённого массива объектов  
6.2. Проверка прав доступа  
6.3. Вывод статуса операции

------------------------------------------------------------
7. Измерение производительности (module timing)
------------------------------------------------------------
7.1. Засечь время загрузки JSON  
7.2. Засечь время поиска  
7.3. Засечь время замены  
7.4. Вывод суммарного времени

------------------------------------------------------------
8. Архитектура проекта (многофайловая)
------------------------------------------------------------
Файлы:
- main.cpp
- generator.hpp/cpp
- json_loader.hpp/cpp
- finder.hpp/cpp
- replacer.hpp/cpp
- reporter.hpp/cpp
- writer.hpp/cpp
- timing.hpp/cpp
- utils.hpp/cpp (если понадобится)

Папки:
src/
include/
data/

------------------------------------------------------------
9. Возможность расширения (плагины, DLL)
------------------------------------------------------------
9.1. Вынести функции поиска в отдельную DLL  
9.2. Вынести функции замены в отдельную DLL  
9.3. Оставить интерфейс IPlugin { run(FileObject&) }  
9.4. Динамическая загрузка DLL через LoadLibrary()

------------------------------------------------------------
10. Будущее GUI приложение
------------------------------------------------------------
- Qt Widgets или Qt Quick  
- Окно выбора JSON-файла  
- Поля: строка поиска, строка замены  
- Чекбоксы: ignore case, dry-run  
- Таблица отчёта  
- Кнопка "Выполнить"  
- Лог ошибок  
- Прогресс-бар для больших файлов
